Captura leads do seu site para o Talke CRM. Inclui tracker, Elementor e WooCommerce.

== Description ==

O plugin Talke CRM conecta seu site WordPress ao [Talke CRM](https://crm.talke.com.br), capturando leads automaticamente. Recursos:

* Instalação do tracker em todas as páginas
* Captura nativa de formulários do Elementor Pro
* Captura de novos pedidos e cadastros do WooCommerce
* Conexão "OAuth-like" sem precisar copiar tokens manualmente

== Installation ==

1. Instale e ative o plugin
2. Vá em "Configurações > Talke CRM"
3. Clique em "Conectar com Talke CRM"
4. Autorize no CRM e escolha qual conta/cliente vincular

== Frequently Asked Questions ==

= Preciso ter conta no Talke CRM? =

Sim. Crie uma em https://crm.talke.com.br.

= O plugin funciona sem Elementor ou WooCommerce? =

Sim. Sem eles, o plugin só injeta o tracker (que ainda captura via auto-detect JS).
